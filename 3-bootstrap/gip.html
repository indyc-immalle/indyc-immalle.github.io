Webapplicatie

Implementatie

Voor de back-end van mijn webapplicatie ga ik vooral SQLite en Flask gebruiken omdat ik deze code talen al geleerd heb op school. Voor de front-end ga ik vooral HTML/CSS gebruiken omdat ik deze markup talen al goed beheers. In de toekomst kunnen er nog andere code talen toevoegd worden. De nadruk van mijn applicatie ligt vooral op de back-end omdat ik deze wil gebruiken bij toekomstige projecten.

Als u op onderstaande knop drukt vindt u een ER-diagramma en een database-schema van mijn webapplicatie.
